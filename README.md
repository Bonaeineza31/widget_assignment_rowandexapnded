# Row + Expanded  Widget Tryout(Flutter)

A tiny Flutter demo showing how `Expanded` makes widgets inside a `Row` share limited space and prevents overflow.

## Run
1. Clone the repo
2. Run:
   ```bash
   flutter pub get
   flutter run device
   flutter run -d emulator-5554

* With flutter pub get you install all packages
* Flutter run device , you will see all available devices that are connected
* then run on the emulator, make sure qre prequirisites are installed beforehand
